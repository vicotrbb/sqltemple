## macOS cannot open the app

If macOS warns the app is from an unidentified developer, right-click the app and choose **Open**. Future builds will be notarized.

## Connection fails

- Verify host/port and that your IP is allowlisted.
- Try enabling SSL if required by your provider.
- Check firewall/VPN rules.

## AI responses are slow

- Use `gpt-4o-mini` for speed.
- Reduce context by limiting schema scope.
- Confirm your API key has quota.

## Editor feels sluggish

- Disable unused tabs.
- Limit result set size when previewing huge tables.
