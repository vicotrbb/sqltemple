## Execute queries

- **Run selection**: highlight text and press `Cmd+Enter` (macOS) / `Ctrl+Enter` (Windows/Linux).
- **Run file**: press `Cmd+Enter`/`Ctrl+Enter` with no selection.
- Results appear in the lower panel with virtualization for large datasets.

## Plans and optimization

- Click **Explain** to view the execution plan tree with timing metrics.
- Use **AI Optimize** to get suggestions tailored to the plan and schema.

## History and tabs

- Each run is added to query history; reopen items from the history panel.
- Tabs support renaming and reordering; close with `Cmd+W` / `Ctrl+W`.

## Export

- Export results to CSV, JSON, or Excel directly from the results grid.
